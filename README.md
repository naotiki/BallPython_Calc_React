## モルフの名前について
モルフの名前はそのまま組み合わせて出力しています。

例. モハべ + ミスティック = モハべ ミスティック(正しくはミスティックポーション)

これを正しくするには、出力結果の表にあるペンのボタンをクリックすることで変更のリクエストを送れます。
